About MMO of Kilimanjaro

----------------------------------------------------------------------------------------------------


This fictional and non-existent MMO called "MMO of Kilimanjaro" is a survival

game. This survival game has a market, and this market is represented on this

website. This market will contact a back-end AWS server to retrieve, add, and

remove data to and from itself.


Works kind of the same way as doing the back-end through the Universities

California servers with mySQL. Only, with AWS you don't have to code the back

-end as AWS does all the coding for the database for you so that you can

mainly focus on front-end coding and technologies.


What this market consists of is a bunch of items that carry the following

information: An image, item name, item price, seller name, location of seller

(in the game), quantity of items, taxes, and item description.


So, the table essentially looks like this:


Item Id Image Item Name Item Price Item Description Seller Name Location
0000000 https://www.example.com/ Sample 19 -------------------- -------------------- --------------------

The image will just be a url or a pointer to an image in the image folder at

img/image_name.XXX, the url can be shrunk down with https://tinyurl.com/ as

the table has limited space. The other data such as taxes and quantity of

items will be calculated in this front-end with javascript functionality.
